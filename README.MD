
# Cash Money Management

## Deskripsi
Aplikasi Cash Money Management adalah solusi manajemen keuangan pribadi yang dirancang untuk membantu pengguna melacak, menganalisis, dan mengelola pengeluaran serta pemasukan uang tunai mereka dengan mudah dan efisien.

## Fitur Utama
- 💰 Pencatatan transaksi (pemasukan dan pengeluaran)
- 📊 Laporan dan analisis keuangan
- 🏷️ Kategorisasi pengeluaran
- 📈 Visualisasi data statistik
- 💾 Penyimpanan data lokal
- 🔍 Pencarian dan filter transaksi
